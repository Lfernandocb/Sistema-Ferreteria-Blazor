# Sistema de Gestión para Ferretería

Proyecto desarrollado en ASP.NET Blazor y SQL Server.

## Tecnologías utilizadas
- ASP.NET Blazor
- SQL Server
- Bootstrap
- Entity Framework Core

## Funcionalidades
- Inicio de sesión por roles
- Gestión de inventario
- Facturación
- Kardex de productos
- Control de caja
- Reportes de ventas
- Búsqueda y paginación
- CRUD de productos

## Roles del sistema
- Administrador
- Vendedor
- Cajero
